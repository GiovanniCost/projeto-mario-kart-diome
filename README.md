# Mario Kart! 🏁

## Sobre o Jogo

Prepare-se para uma corrida dinâmica onde dois personagens, cada um em seu veículo, disputarão a vitória em uma pista gerada aleatoriamente. Cada rodada traz um novo desafio, exigindo estratégia e um pouco de sorte!

## Regras do Jogo

### 🎮 Jogadores

A corrida será disputada entre dois personagens controlados pelo computador. Cada personagem possui um conjunto de atributos que influenciarão o desempenho nas diferentes partes da pista.

### 🛣️ Pistas

*   A cada partida, uma pista aleatória de **5 rodadas** é gerada.
*   Em cada rodada, um novo "bloco" da pista é sorteado, podendo ser:
    *   **RETA** 🚀
    *   **CURVA** ↪️
    *   **CONFRONTO** 💥

### 🎲 Desafios das Rodadas

A cada tipo de bloco de pista, um desafio diferente acontece:

1.  **RETA (🚀):**
    *   Os jogadores jogam um dado de 6 lados.
    *   O valor do dado é somado ao atributo **VELOCIDADE** de cada personagem.
    *   O jogador com a maior pontuação total (dado + velocidade) nesta rodada ganha **1 ponto**.

2.  **CURVA (↪️):**
    *   Os jogadores jogam um dado de 6 lados.
    *   O valor do dado é somado ao atributo **MANOBRABILIDADE** de cada personagem.
    *   O jogador com a maior pontuação total (dado + manobrabilidade) nesta rodada ganha **1 ponto**.

3.  **CONFRONTO (💥):**
    *   Os jogadores jogam um dado de 6 lados.
    *   O valor do dado é somado ao atributo **PODER** de cada personagem.
    *   O jogador com a **menor** pontuação total (dado + poder) nesta rodada **perde 1 ponto**.
    *   **Importante:** Nenhum jogador pode ter pontuação negativa; a pontuação mínima é 0.

### 🏆 Condição de Vitória

Ao final das 5 rodadas, o personagem que acumulou o maior número de pontos é declarado o grande vencedor da Super Corrida Maluca!

---
Que vença o melhor... ou o mais sortudo! 😉
